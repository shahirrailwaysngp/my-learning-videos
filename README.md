यह पोर्टल लोकोमोटिव कैमरा से ली गई सेक्शनल रोड वीडियो सिमुलेशन के लिए बनाया गया है। इसमें सभी महत्वपूर्ण स्टेशनों, सिग्नलों, ग्रेडिएंट्स और खतरनाक जगहों की वीडियो उपलब्ध हैं।
