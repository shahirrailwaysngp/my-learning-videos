# SECR Nagpur Division - E-Learning Simulation
## SECTIONAL ROAD LEARNING VIDEOS
**SOUTH EAST CENTRAL RAILWAY | NAGPUR DIVISION**

Electric(Op) Department _ Nagpur heartily welcomes you.

Description :-

This portal has been created for sectional road video simulation using footage captured from locomotive cameras. It includes videos of all important stations, signals, gradients, and hazardous locations.

Disclaimer :-

All these videos are intended for educational purposes only. They should not be used as a substitute for actual train operation training.

Key Features:-
Detailed simulation of all sectional roads
Understanding signal identification and speed control
Identification of gradients and curves
Information about hazardous locations
Bilingual support (Hindi + English)
Works on both mobile and desktop devices
